安装说明：

1.首先访问你的网站 http://你的域名/install/   根据提示完成数据库和站长信息的配置安装。

2.然后打开根目录下的 config.php 完成基本配置，注意一定要修改网站密钥，同时cache,uploads的文件夹设置777权限。

3.安装完成请删除install文件夹
